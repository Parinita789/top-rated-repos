## Top Rated Respositories

## Description
Top Rated Respositories fetches top rated repositories from different data sources.

## Installation

```bash
$ npm install
```

## Configuration 

create development.yml file in config folder using default.yml file

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
## Running the app with docker

```bash
docker compose up
```


## Test

```bash
# e2e tests
$ npm run test:e2e
```
